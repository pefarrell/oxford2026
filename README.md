# Summary

This repository contains the slides and code for a lecture course given by

Patrick E. Farrell <patrick.farrell@maths.ox.ac.uk>

on

Solving partial differential equations with Firedrake

at the University of Oxford in March 2026. See

https://www.firedrakeproject.org/tutorial_mar_26.html

for details.

# Usage

Most participants used Google Colab to code with Firedrake without having to
install it. To use Colab, go to

https://colab.research.google.com

and then paste the code in colab.txt into a cell.
